# MuseDash Editor

An unofficial MuseDash chart editor for [MDMC](https://mdmc.moe/).

## Project Status

This project is in early (but active) development.

It follows a simple guideline: "make it exist first, we can make it better later." Thus, the editor is really ugly currently. I'm a dev, not a designer :D.

You can find the remaining tasks to do in the [project board](https://github.com/orgs/MuseDashEditor/projects/1), when I have time to keep it up-to-date.

## Contributing

Contributions are welcome, but with a few restrictions:
1. No AI-generated code. You can use auto-completion for instance, but not claude, copilot etc.
2. Please ask before doing anything to avoid having multiple people working on the same tasks
3. Please refrain from refactoring big chunks of code without permission
4. I know that I don't really do it myself, but please try to write tests when adding things
5. Keep your changes simple, if you change many parts of the code, split them into multiple PRs

> [!IMPORTANT]  
> Due to copyright reasons, the assets used in the editor are not included in this repository, and belong to their respective owners.
> If you want to build the editor yourself, please contact a maintainer to get access to the private repository.
