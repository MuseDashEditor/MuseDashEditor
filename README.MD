# MuseDash Editor

An unofficial MuseDash chart editor for [MDMC](https://mdmc.moe/).

## Project Status

This project is in early development.

It follows a simple guideline: "make it exist first, we can make it better later." Thus, the editor is really ugly currently. I'm a dev, not a designer :D.

You can find the remaining tasks to do in the [project board](https://github.com/users/Azn9/projects/3/views/1), when I have time to keep it up-to-date.
